Mais Formatações

TEXTO MARCADO

Podemos criar também um texto marcado usando a tag <mark> (tag semântica).

TEXTO GRANDE E PEQUENO

Estamos criando um texto grande (tag <big>) e um texto pequeno (tag <small>) nesse parágrafo.

TEXTO DELETADO

Podemos marcar um texto como excluido usando a tag <del>.
Indicando que ele deve ser lido, mas não considerado.

TEXTO INSERIDO

Podemos marcar um texto como inserido usando a tag <ins>.
Indicando que foi adicionado depois.

Também podemos sublinhar utilizando a tag <u> (tag não semântica).

TEXTO SOBRESCRITO

Utilizado em casos para inserir coisas do tipo x20+3, usando a tag <sup>.

TEXTO SUBSCRITO

Utilizado em casos para inserir coisas do tipo H2O, usando a tag <sub>.
